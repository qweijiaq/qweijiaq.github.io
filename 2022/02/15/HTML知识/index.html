HTML简介

1.什么是HTML

HTML，Hyper Text Markup Language，超文本标记（标签）语言
由各种标签组成，用来制作网页，告诉浏览器该如何显示网页

2.作用

  • 制作网页，控制网页和内容显示
  • 插入图片、音乐、视频、动画等多媒体
  • 通过链接检索信息
  • 使用表单获取用户信息，实现交互

    3.版本

    W3C，World Wide Web Consortium万维网联盟，制定Web技术相关标准和规范的组织，HTML就是W3C制定的标准
    两个版本：HTML4.01，HTML5
    官网

    4.后缀名

    HTML文件是以.html或.htm结尾

HTML文档结构

1.基本结构

1.1 简介

  • HTML标签是由尖括号包围的关键词，如<html>，通常都是成对出现的，如<html></html>
  • 以<html>为根标签，包含<head>头部、<body>主体部分
  • 头部提供网页的相关信息，如标题、文档类型、字符编码等摘要信息
  • 主体部分提供网页的具体内容，真正显示在页面中
  • 合理的进行缩进
  • 标签名不区分大小写，但一般都习惯小写

1.2 浏览器

常见浏览器：IE微软、Chrome谷歌、Firefox火狐、Safari苹果

浏览器的作用是读取HTML文件，并以网页的形式显示

浏览器不会直接显示HTML标签，而是使用标签来解释网页的内容

2.标签

2.1 标签组成

一个完整的HTML标签的组成：

<标签名 属性名=“属性值”>内容</标签名>

属性值要用引号引起来，一般使用双引号

2.2 标签的分类

  • 根据标签是否关闭，分为：关闭型、非关闭型

    • 关闭型：有结束标签，即成对出现

      <title></title>
<body></body>
<h1></h1>
...

    • 非关闭型：没有结束标签

      <meta>
<br>
<hr>
...

  • 根据标签是否独占一行，分为：块级标签、行级标签

    • 块级标签：显示为块状，独自占一行

      <h1></h1>
<div></div>

    • 行级标签：在行内显示，可以与其它内容在同一行显示

      <p></p>
<span></span>

3.注释

注释在浏览器中是不会显示的，用来标注解释，但可以通过源代码查看

<!-- 注释内容 -->

4.实体字符

也称为特殊字符，用于显示一些特殊符号

语法：

&实体字符名称;

常用实体字符：

&lt;		小于号
&gt;		大于号
&nbsp;		空格
&amp;		与&
&quot;		双引号
&copy;		版权
&reg;		注册符号
&times;		关闭符号，乘号

注：实体字符名称是严格区分大小写的

对于连续的空白字符（包括空格、缩进、换行等），在浏览器中显示时只会显示为一个空格。

5.文档类型

在HTML文档的第一行，使用<!DOCTYPE>声明HTML文档的类型

用来告诉浏览器页面的文档类型，简单来说，用来指定使用的HTML的版本规范

目前基本上都使用HTML5：<!DOCTYPE html>

br 换行标签 非关闭型

h1,h2,…,h6 标题标签 关闭型，块级标签，加粗显示，1-6逐级变小，h7及之后无效果

1.有序列表

默认使用阿拉伯数字，从1开始标记，可以通过属性进行修改

  • type属性：设置列表前的符号标记，取值：数字1、字母a或A、罗马数字i或I
  • start属性：设置起始值，值必须是数字

2.无序列表

默认使用实心圆作为符号标记，可以通过属性进行修改

  • type属性：设置列表前的符号标记，取值：disc实心圆（默认）、circle空心圆、square正方形、none不显示符号

3.定义列表

dl：definition list

dt：definition title

dd：definition description

4.水平线标签

hr：horizontal

常用属性：

  • color：颜色

    • 两种写法：
      • 颜色名称，如red、green、blue、white、black、pink、orange等
      • 16进制的RGB：#RRGGBB，每个颜色的范围为00-FF(0-255)，如#FF0000、#CCCCCC

  • size：粗细，使用数值

  • width：宽度
    • 像素：绝对值（固定值）
    • 百分比：相对值，相对于父容器宽度的百分比
  • align：对齐方式
    • center：居中（默认）
    • left：居左
    • right：居右

5.图像标签

img：image

常见图片格式：jpg、png、gif、bmp

常用属性：

  • src：source，指定图片的来源（路径），必须有
  • alt：当图片无法显示时的提示信息
  • title：当鼠标悬停时的提示信息
  • width、height：宽和高
    • 默认图片原尺寸显示
    • 如果只设置其中的一个，则另一个按原比例缩放
    • 如果同时设置宽和高，可能会导致图片变形
    • 两种写法：
      • 像素：绝对值（固定值）
      • 百分比：相对值，相对于父容器的百分比

浏览器渲染后显示的页面代码可能与编码时有所不同

浏览器提供的开发工具：用于帮助开发人员查看和调试页面

  • Elements：从浏览器的角度查看页面，浏览器渲染页面时的结构和内容
  • Console：控制台，显示各种警告和错误信息
  • Network：查看网络请求信息，浏览器向服务器请求了哪些资源、资源大小、加载资源花费的时间等

超链接

1.简介

使用超链接可以从一个页面跳转到另一个页面，实现页面间的导航

当鼠标移动到超链接文本上时，鼠标箭头会变成一只小手

超链接有三种类型：

  • 普通链接/页面间链接，跳转到另一个页面
  • 锚链接，跳转到锚点
  • 功能性链接，实现特殊功能

2.基本用法

使用<a>标签创建超链接

语法：

<a href="链接地址">链接文本或图像</a>

常用属性：

  • href：链接地址/路径，链接地址
    • 路径分类：
      • 相对路径：相对于当前文件的路径
        • .表示当前路径
        • ..表示上一级路径
      • 绝对路径：以根开始的路径
  • target：链接打开的位置
    • _self：当前页面打开，默认
    • _blank：新建页面打开
    • _parent：父层框架中打开
    • _top：顶层框架中打开
    • 自定义

3.锚链接

3.1 简介

点击链接后跳转到指定的位置（锚点anchor）

锚链接的分类：

  • 页面内的锚链接
  • 页面间的锚链接

3.2 页面内的锚链接

步骤：

  1. 定义锚点（标记）

    <a name="锚点位置">目标位置</a>

  2. 链接锚点

    <a href="#锚点位置">链接文本</a>

3.3 页面间的锚链接

步骤：

  1. 定义锚点（标记）

    <a name="锚点位置">目标位置</a>

  2. 链接锚点

    <a href="目标页面#锚点位置">链接文本</a>

4.功能性链接

<!-- 发邮件 -->
<a href="mailto:xxx@qq.com">联系我们</a>
<!-- 运行脚本 -->
<a href="javascript:alert('嘿嘿')">点我试试</a>
...

5.URL

5.1 简介

URL：Uniform Resource Locator统一资源定位符，用来定位资源所在位置，最常见的就是网址

https://www.baidu.com
http://www.baidu.com
ftp://ftp.itany.com/note/HTML.md
...

5.2 组成

一个完整的URL由8个部分组成：

  • ==协议==protocol，如：
    • http：超文本传输协议，用来访问Web网站 Hyper Text Transfer Protocol
    • https：加密的http协议
    • ftp：文件传输协议，用来访问服务器上的文件，实现文件的上传和下载 File Transfer Protocol
    • file：文件协议，用来访问本地文件
  • ==主机名== hostname 服务器地址，如www.baidu.com、ftp.itany.com
  • ==端口== port 位于主机名的后面，使用冒号隔开
    • 不同的协议使用不同的端口号，如http使用80，ftp使用21，https使用443
    • 如果使用的是默认端口，则端口号可以省略
    • 如果使用的不是默认端口，则必须指定端口
  • ==路径== path 目标文件所在的路径文件，如/web/html/imgs
  • ==资源== resource 要访问的目标文件，如01.png
  • ==查询字符串== query string 也称为参数
    • 在资源后面使用?开头的一组名称组
    • 名称和值之间以=隔开，多个之间以&隔开，如?a=1&b=2&c=3
  • ==锚点== anchor 在资源后面使用#开头的文本，如#first
  • ==身份认证== authentication 指定身份信息，如：ftp://账号:密码@ftp.itany.com/note/HTML.md

表格

1.简介

表格是一个规则的行列结构，每个表格由若干行组成，每行由若干个单元格组成

table、row、column

2.基本结构

2.1 table标签

用来定义表格

常用属性：

  • border：边框，默认为0
  • width、height：宽和高
  • align：对齐方式
    • center：居中
    • left：居左，默认
    • right：居右
  • bordercolor：边框颜色
  • bgcolor：背景颜色
  • background：背景图片
  • cellspacing：间距，单元格与单元格之间的距离
  • cellpadding：边距，单元格之中的内容与边界的距离

2.2 tr标签

table row 用来定义行

常用属性：

  • align：水平对齐方式
    • center：居中
    • left：居左，默认
    • right：居右
  • valign：垂直对齐方式
    • middle：居中
    • top：顶部对齐
    • bottom：底部对齐
  • bgcolor：背景颜色
  • background：背景图片

2.3 td标签

table data 用来定义单元格

常用属性：align、valign、bgcolor、background

表格必须由行组成，行必须由单元格组成，数据必须放在单元格td里

3.合并单元格

也称为表格的跨行跨列

两个属性：

  • rowspan：设置单元格所跨的行数，如：rowspan=2表示跨越两行
  • colspan：设置单元格所跨的列数，如：colspan=2表示跨越两列

步骤：

  • 在跨越的单元格中设置rowspan/colspan属性
  • 将被跨越的单元格删除

4.高级标签

4.1 caption标签

表格的标题

4.2 thead标签

表格的头部

4.3 th标签

表格的头部标题

一般用在thead标签中，设置头部的标题，代替td标签

显示的效果为：居中 + 加粗

4.4 tbody标签

表格的主体

4.5 tfoot标签

表格的底部

表单

1.简介

表单是一个包含若干个表单元素的区域，用于获取不同的类型的用户信息

表单元素是允许用户在表单中输入信息的元素，如：文本框、密码框、单选按钮、复选框、下拉列表、按钮等

2.结构

2.1 表单语法

<form>
    多个表单元素
</form>

2.2 form标签

用来定义表单，可以包含多个表单元素

常用属性：

  • action：提交数据给谁处理，即处理数据的程序
  • method：提交数据的方式
    • get：以查询字符串的形式提交，数据在地址栏中能看到，且长度有一定的限制，不安全
    • post：以表单数据组形式提交，在地址栏中看不到，长度无限制，安全
  • enctype：提交数据的编码
    • application/x-www-form-urlencoded，默认，可不写
    • multipart/form-data，==文件上传时使用==

3.表单元素

大多数表单元素都是使用input标签来定义的，通过设置type属性来定义不同的表单元素

表单元素类型 含义 说明
text 单行文本框 省略时默认为text
password 密码框 输入时以点号显示，安全
radio 单选按钮 只能选择其中一个
checkbox 复选框 可以同时选择多个
submit 提交按钮 提交表单数据
reset 复位按钮 重置表单，使其恢复为初始值
image 图像按钮 可以使用一张图片作为按钮，可以提交
button 普通按钮 默认没有功能
file 文件选择器 选择要上传的文件，==需要更改enctype属性==
hidden 隐藏域 在页面上不显示，但会提交，可以用来存储数据 
<input type="表单元素类型" name="名称">

3.1 单行文本框

常用属性：

  • name：名称，==很重要==，如果没有指定name，则该表单元素的数据是无法提交的
  • value：初始默认值
  • size：指定宽度
  • maxlength：最大字符数，默认没有限制
  • readonly：只读（不可更改，但可提交），readonly="readonly"可简写为readonly，即只写属性名
  • disabled：禁用（完全禁用，不可更改，也无法提交），disabled="disable"可简写为disabled

表单元素被提交的两个条件：

  1. 有name属性
  2. 非disabled

3.2 单选按钮

常用属性：

  • name：名称，==很重要==，必须指定name属性才能实现单选的互斥，否则可以同时选择
  • value：值
  • checked：是否选中
    • 设置checked="checked"来默认选中，可简写为checked

3.3 复选框

常用属性和单选按钮类似

3.4 文件选择器

常用属性：

  • name：名称
  • accept：指定接受文件的类型，使用MIME格式字符串对资源类型进行限制
    • 纯文本：text/plain text/html text/css text/javascript …
    • 图像：image/jpeg image/png image/gif …
    • 音频：audio/* audio/mpeg audio/wav …
    • 视频：video/* video/webm video/ogg …
    • 二进制数据：application/* application/xml、application/pdf …

4.特殊表单元素

表单元素 含义 说明
select 下拉列表
option 列表选项
optgroup 选项组 用来对option进行分组
textarea 文本域、多行文本框 用来创建多行文本框

4.1 下拉列表

select常用属性：

  • name：名称
  • size：行数，同时显示多个选项
  • multiple：允许同时选择多个选项
  • disabled：禁用

option常用属性：

  • value：值
  • selected：默认选中项
<form action="">
    学历：
    <select name="degree">
    	<option value="0">请选择学历</option>
    	<option value="1">大专</option>
    	<option value="2">本科</option>
    	<option value="3">硕士</option>
    	<option value="4">博士</option>
	</select>
</form>

optgroup常用属性：

  • label：分组的标题

4.2 文本域

常用属性：

  • name：名称
  • rows：行数
  • cols：指定宽度

文本域标签中的内容会按原格式显示，类似与<pre>标签的效果。

5.其他表单元素

5.1 label标签

为表单元素提供标签，当选中label标签中的文本内容时会自动将焦点切换到与之关系的表单元素

常用属性：

  • for：必须将该属性值设置为与之关联表单元素的id属性相同

几乎所有HTML标签都具有id属性，且==id值必须唯一==。

<form action="">
    <label for="name">用户名：</label>
    <input type="text" name="username" id="name">
</form>

5.2 button标签

也表示按钮，与input按钮类似

语法：

<button type="按钮类型">
    按钮文本或图像
</button>

常用属性：

  • type：按钮的类型
    • submit：提交（默认）
    • reset：重置
    • button：普通按钮
<input type="submit" value="提交按钮">
<input type="reset" value="重置按钮">
<input type="image" src="imgs/submit.gif">
<input type="button" value="普通按钮">

<button  type="submit">提交按钮</button>
<button  type="reset">重置按钮</button>
<button><img src="imgs/submit.gif"></button>
<button type="button">普通按钮</button>

5.3 fieldset和legend标签

fieldset标签用于对表单的元素进行分组

legend标签用于对分组添加标题

<form>
    <fieldset>
        <legend>院校信息</legend>
        学号：<input type="text" name="stuId"><br>
        学校：<input type="text" name="stuSchool"><br>
        专业：<input type="text" name="stuMajor"><br>
    </fieldset>
</form>

内嵌框架

1.简介

使用iframe可以在一个页面中引用另一个页面，实现复用

使用灵活

2.基本用法

语法：

<iframe src=""></iframe>

常用属性：

  • src：引用的页面
  • width/height：指定宽度/高度，像素或百分比
  • frameborder：是否显示边框，取值：
    • 1(yes)：显示边框（默认）
    • 0(no)：不显示边框
  • scrolling：是否显示滚动条，取值：yes、no、auto
  • name：名称

3.在框架中打开链接

将a标签的target属性和iframe标签的name属性设置为相同

<iframe name="hello"></iframe>
<a href="链接地址" target="hello">链接文本或图像</a>

HTML5简介

1.发展

W3C于1992.12发布HTML4.01标准

W3C于2014.10发布HTML5标准

2.特点

  • 取消了过时的标签，如：font center ，它们仅用于展示外观，可通过CSS实现
  • 增加了一些更具有语义化的标签，如header footer aside等
  • 增加了一些新功能标签，如audio video canvas等
  • 增加了一些表单控件，如email date time url search等
  • 可以直接在浏览器中绘画(canvas)，无需flash
  • 增加了本地存储的支持

3.兼容性

可通过http://caniuse.com查看标签在各种浏览器中是否可用

提供了各浏览器版本对HTML5和CSS3规范的支持度

HTML5新增内容

1.结构相关标签

用来进行页面结构布局，本身无任何特殊样式，需要使用CSS进行样式设置

  • article：定义一个独立的内容、完整的文章
  • section：定义文档的章节、段落
  • header：定义文章的头部、页眉、标题
  • footer：定义文章的底部、页脚、标注
  • aside：定义侧边栏
  • figure：定义图片区域
  • figcaption：为图片区域定义标题
  • nav：定义导航栏菜单
<article>
	<header>标题</header>
    <section>第一段</section>
    <section>第二段</section>
    <footer>脚注</footer>
</article>

<aside>广告</aside>

<figure>
    <figcaption>Logo标识</figcaption>
	<img src="" alt="">
    <img src="" alt="">
</figure>

<nav>
	<ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</nav>

结构标签只是表明各部分的角色，并无外观样式，与普通div标签相同。

2.语义相关标签

2.1 mark标签

标注，用来突出显示（高亮）文本

2.2 time标签

定义日期和时间，便于搜索引擎智能查找

<time datetime="2022-02-14">情人节</time>

2.3 details和summary标签

默认显示summary中的内容，点击后显示detail中的内容

<details>
	<summary>HTML简介</summary>
    <p>HTML是一门用来制作网页的标签语言</p>
</details>

并不是所有浏览器都兼容，Chrome、Opera支持，Firefox不支持

2.4 meter标签

计数仪，表示度量

常用属性：

  • max：定义最大值，默认为1
  • min：定义最小值，默认为0
  • value：定义当前值
  • high：定义限定为高的值
  • low：定义限定为低的值
  • optimum：定义为最佳的值

规则（计数仪颜色表示，绿色为优，红色为劣，黄色为一般）：

  1. 如果optimum大于high，则表示值越大越好
    1. 当value大于high时为绿色
    2. 当value在high和low之间为黄色
    3. 当value小于low为红色
  2. 如果optimum小于low，则表示值越小越好
    1. 当value大于high时为红色
    2. 当value在high和low之间为黄色
    3. 当value小于low为绿色
  3. 当optimum介于low和high之间，则表示值在low和high之间比较好
    1. 当value大于high或者小于low均为黄色
    2. 当value在high和low之间为绿色

2.5 progress标签

进度条，表示运行中的进度

常用属性：

  • value：定义当前值
  • max：定义完成的值

3.表单相关

3.1 新增表单元素

新增以下type类型：

  • email：接收邮箱，不符合邮箱格式的无法提交
  • url：接收链接，不符合链接格式的无法提交
  • tel：接收电话号码，目前仅在移动设备中有效
  • search：搜索文本框，效果上有一个叉号，可以直接清空输入的内容
  • number/range：接收数字/数字滑块包含min（最小值）、max（最大值）、step（步长）属性
  • date/month/week/time/datetime：日期时间选择器，兼容性不好
  • color：颜色拾取

作用：

  • 具有格式校验的功能
  • 可以与移动设备的键盘相关联

3.2 新增表单属性

form标签的属性：

  • autocomplete：是否启用表单的自动完成功能，取值：on（默认）、off，前期搜索或输入的内容会保存下来，后期输入时可以直接选择补充
  • novalidate：提交表单时不进行校验，默认会进行校验

3.3 新增表单元素属性

input/select/textarea等

  • placeholder：提示文字，文本框内的提示内容，如邮箱文本框中的“请输入邮箱”提示字样

  • required：设置必填项

  • autocomplete：是否启用表单元素的自动完成功能

  • autofocus：设置初始焦点元素

  • pattern：使用正则表达式(RegExp)进行数据校验

  • list：使文本元素具有下拉列表的功能，需要配合datalist和options标签使用

    <label for="city">城市：</label>
<input type="text" name="city" id="city" list="cityList">
<datalist id="cityList">
	<option value="nanjing">南京</option>
    <option value="beijing">北京</option>
    <option value="hefei">合肥</option>
</datalist>

    将input的list属性和datalist的id属性设为相同。

    ==list和普通下拉列表的区别：普通的下拉列表只能选择，list属性形成的下拉列表不仅能选择还能输入没有的选项进行提交。==

  • form：可以将表单元素写在form标签外面，然后通过该属性来关联指定的表单

4.多媒体相关

4.1 audio标签

在页面中插入音频，不同的浏览器对音频格式的支持不一样

audio常用属性：

  • src：音频文件的来源
  • controls：是否显示控制面板，默认不显示，直接加上controls
  • autoplay：是否自动播放，默认不自动播放，直接加上autoplay
  • loop：是否循环播放，默认不循环，直接加上loop
  • muted：是否静音，默认不静音，直接加上muted
  • preload：是否预加载，取值：none（不预加载）、auto（预加载，默认）、metadata（只预加载元数据）
    • 如果设置了autoplay，则该属性无意义

可以结合source标签使用，指定多个音频文件，浏览器会检测并使用第一个可用的音频文件，有效缓解audio属性无法识别src音源的问题

4.2 video标签

在页面中插入视频，不同的浏览器对视频格式的支持不一样

用法与audio标签基本相同，增加属性：

  • width/height：设置视频播放器的宽度/高度
  • poster：设置视频加载前显示的图片
